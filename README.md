# s21SimpleNavigator

Библиотека для работы с графами и реализация алгоритмов поиска пути в графе.

## Описание

Проект реализует основные алгоритмы на графах, используя структуру данных матрица смежности. Включает в себя консольное приложение для демонстрации работы алгоритмов.
Особенностью проекта является использование C++ контейнеров (Queue, Stack) через CGO для реализации некоторых алгоритмов.

## Функциональность

- **Загрузка графа из файла**: Поддержка текстового формата матрицы смежности.
- **Обход графа**:
  - Поиск в ширину (BFS) - с использованием очереди.
  - Поиск в глубину (DFS) - с использованием стека.
- **Поиск кратчайшего пути**:
  - Алгоритм Дейкстры (для поиска пути между двумя вершинами).
  - Алгоритм Флойда-Уоршелла (для поиска путей между всеми парами вершин).
- **Построение минимального остовного дерева**:
  - Алгоритм Прима.
- **Решение задачи коммивояжера (TSP)**:
  - Муравьиный алгоритм (Ant Colony Optimization).

## Структура проекта

- `cmd/client/`: Точка входа в приложение (консольный интерфейс).
- `pkg/algorithms/`: Реализация алгоритмов на графах.
- `pkg/graph/`: Структуры данных графа и функции загрузки.
- `pkg/containers/`: Go-обертки над C++ контейнерами (CGO).
- `pkg/cpp_containers/`: Реализация контейнеров на C++.

## Требования

- Go 1.20+
- GCC / Clang (для компиляции C++ кода через CGO)
- Make

## Сборка и запуск

Для сборки и запуска проекта используйте `make`:

```bash
make run
```

Очистка скомпилированных файлов:

```bash
make clean
```

Запуск тестов:

```bash
make test
```

### Формат файла графа

Файл должен содержать матрицу смежности.
- Первая строка: количество вершин (N).
- Следующие N строк: N чисел, разделенных пробелами, представляющих веса ребер.

Пример:
```
3
0 1 5
1 0 2
5 2 0
```
